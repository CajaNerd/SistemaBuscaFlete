Pasos Para Postulación

1.Presiona en Postular Gratis Aqui

2. Completa y Envia el Formulario Con toda la información Solicitada.

3. Te debera llegar en unos minutos un correo con la información previa de tu futura cuenta

4. Descarga e instala la aplicacion movil exclusiva para Choferes desde tu pagina de inicio.

5. Si toda tu documentación es correcta nos comunicaremos contigo para felicitarte, tenemos un plazo minimo de 2 dias habiles para comprobar tus datos.

6. ¡Listo! ya estas listo para recibir solicitudes

Procesos Basicos de Usuarios en buscaflete.cl

Cliente

1. Registro Cliente

2. Inicio de Sesión Cliente

3. Cliente sube a lo menos una fotografia para que tu puedas identificar la carga mas adelante

4. Cliente ingresa el destino

5. Cliente Revisa el MAPA (Rutear)

6. Cliente Envia Solicitud

7. Cliente Acepta/Rechaza precio propuesto

8. Cliente Finaliza Solicitud

9. Cliente Califica Chofer

Chofer

1. Registro Chofer

2. Inicio de Sesión Chofer

3. Chofer recibe solicitud

4. Chofer envia precio dispuesto a Cliente

5. Chofer realiza el transporte

6. Chofer Califica Cliente
